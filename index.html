Cavern Wars

An iOS game made with XCode 5 for the iPhone 5, based on Age of War

This game is a linear tower defense game.

$ cd your_repo_root/repo_name
$ git clone https://www.github.com/elc1798/CavernWars
$ cd CavernWars

If you do not have 'git' installed, you can use wget:

$ wget https://github.com/elc1798/CavernWars/archive/master.zip
$ unzip master.zip
$ rm master.zip #Optional
